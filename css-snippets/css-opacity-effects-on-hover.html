Opacity - it is the degree to which content behind an element is hidden - this will set the opacity to its content also . which means in this case - On hover opacity will make the element opaque, opacity 0 (fully transparent) will make it disappear completely, 1 (fully opaque)will have no difference whatsoever - opacity can take a value from 0 (fully transparent) to 1 fully opaque or in percentage from 0 to 100% - opacity inherit will inherit the opacity of its parent - opacity initial can be used to reset to it default when the page initially loaded,. - opacity 1 is by default - the problem of using opacity directly on the element will cause the child elements/content inside that element also to be opaque. If you do not want the content inside it to have that effect but just the background color, then instead of applying the opacity to the element, you should apply it to the background color instead - eg: - transition: 0.5s; background: rgba(255, 0, 0, 0.3); the same way if this needs to be applied only for the text then it can be applied to the color of the text eg: color: rgba(255, 0, 0, 0.3); and the transition will be shown on the text instead Transition - On hover the opacity 0.3 or 1 is instantaneous , the effect is gradual/smooth. if we want the effect on/offf of the color to be smooth this can be done using transition.
